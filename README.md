# Live-Session-9-Assignment
This assignment goes through the examples in Unit 9 for analyzing TimeSeries data and Volatility of a S&amp;P Stock

The md file in this repository contains the Unit 9 questions and R code that provides the appropiate answer.

There were two assignments

The first assignment goes through time series data by decomposing and performing seasonal and trend analysis.  The dataset used was the hsales contained in the fpp package
The second assignment examines the volatility of a S&P stock. My assigned stock was AGIO.  

Required Packages
fpp
tseries
